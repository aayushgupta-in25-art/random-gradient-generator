# random-gradient-generator
A simple and interactive Random Gradient Generator built using HTML, CSS, and JavaScript.

## Features

- Generate random gradient colors
- Live background preview
- Copy CSS code with one click
- Responsive design
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Click on either color button to generate a new random color.
2. The background gradient updates automatically.
3. Click on the CSS code block to copy the gradient CSS code.
